# 실기기 QA 체크리스트

배포 전/후 실제 기기에서 손으로 확인하는 항목. (자동 테스트로 잡기 어려운 UI·기기 의존 동작 위주)

## 레이아웃 / Safe Area
- [ ] iPhone(노치/홈인디케이터): 하단 탭바 라벨이 가려지지 않음 (viewport-fit=cover + safe-area)
- [ ] iPhone 상단: 헤더가 상태바/노치에 닿지 않음
- [ ] 작은 화면(SE 등)에서 카드/버튼이 잘리거나 겹치지 않음
- [ ] 가로 스크롤 영역(카테고리 칩, 상황 탭, 덱 캐러셀)이 자연스럽게 스크롤됨
- [ ] 모의시험 진행 화면의 '다음' 버튼이 하단 탭/인디케이터에 가리지 않음

## 다크 모드
- [ ] 라이트/다크/시스템 토글이 즉시 반영되고 새로고침 후에도 유지
- [ ] 다크 모드에서 텍스트 대비가 충분(특히 var(--text-3) 보조 텍스트)
- [ ] 카드/표면/보더 색이 다크에서 깨지지 않음
- [ ] 글씨 쓰기(WritingPad) 잉크가 다크에서 보임

## TTS (음성)
- [ ] 단어/예문/회화 듣기 버튼이 소리를 냄 (일본어·영어 각각)
- [ ] 음성 미설치 기기/브라우저에서 앱이 깨지지 않고 조용히 무시
- [ ] 설정의 재생 속도 변경이 반영됨

## 발음 따라하기 / 알림 (브라우저 의존)
- [ ] Chrome/Edge: 🎤 따라 말하기 동작 + 점수 표시
- [ ] Safari/Firefox: 미지원 시 🎤 버튼이 보이지 않음(숨김)
- [ ] 학습 리마인더: 권한 허용 후 테스트 알림 표시 (iOS PWA는 홈화면 추가 후)

## 학습 플로우
- [ ] 스와이프 분류(스키밍) — 좌/우 스와이프 동작, 카드 위 글씨쓰기 시 카드가 넘어가지 않음
- [ ] 복습(SM-2) — 다시/어려움/좋음/쉬움 채점 후 다음 카드
- [ ] 단어장 검색·정렬·★저장이 즉시 반영
- [ ] 오답노트/복습 라이브러리 진입 및 퀴즈
- [ ] 모의시험 난이도(입문/표준/도전)·회차·타이머·결과·점수 추이
- [ ] 회화 읽음 ✓, 롤플레이 완료 ✓ 가 홈 위젯/목록에 반영
- [ ] 여행 회화집 즐겨찾기 ★ + 복습 퀴즈

## 데이터 / 동기화
- [ ] 진도 백업 내보내기 → 다른 기기에서 가져오기 시 즐겨찾기·읽음·롤플레이·시험이력까지 복원(새로고침 후)
- [ ] 로그인 시 Supabase 동기화(진도) 정상
- [ ] 오프라인(PWA)에서 앱 셸 로드

## 접근성
- [ ] 키보드 Tab 이동 + Enter/Space로 카드 뒤집기 동작
- [ ] 아이콘 버튼에 스크린리더 라벨(aria-label) 존재
- [ ] prefers-reduced-motion 설정 시 과한 애니메이션 감소
